# Å rodo

## Requirements

- Node.js
- MongoDB

## Installation

Install required packages for client and server:

```bash
cd server
npm i
cd ../client
npm i
```

## Usage

To start the **server** open new terminal in root directory and type:

```bash
cd server
npm run start
```

To start the **client** open new terminal in root directory and type:

```bash
cd client
npm run dev
```
